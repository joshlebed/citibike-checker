Citi Bike Station Picker

Set up Siri to check bike and dock availability at your favorite stations.

  1. Find your stations. Search for an address or use your location, then click station markers on the map to add them to your profile.
  2. Mark your primary stations. The ones closest to you. Backup stations only show when primary availability is low.
  3. Generate a config and paste it into a Siri Shortcut. Then say "Hey Siri, Citi bike work docks" to hear availability.

Siri Shortcut Setup

  1. Open the Shortcuts app on your iPhone and tap +
  2. Name it (e.g., "Citi bike work docks")
  3. Add action: Text — paste the copied JSON config into the text block
  4. Add action: Get Contents of URL
    • URL:
    • Method: POST
    • Headers: add key Content-Type with value application/json
    • Request Body: change to File, then tap the field and select Text (the output from step 3)
  5. Add action: Speak Text — set it to speak the result from step 4
  6. Done! Trigger with "Hey Siri, Citi bike work docks"
