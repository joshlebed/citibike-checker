Setting up the Shortcut

  1. Open the Shortcuts app on your iPhone or Mac and tap +
  2. Name it (e.g., "Citi bike work docks")
  3. Add action: Text — paste the copied JSON config into the text block
  4. Add action: Get Contents of URL
    • URL:
    • Method: POST
    • Request Body: change to File, then tap the field and select Text (the output from step 3)
  5. Add action: Speak Text — set it to speak the result from step 4
  6. Done! Trigger with "Hey Siri, Citi bike work docks"
